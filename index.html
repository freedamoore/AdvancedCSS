Hover over the images

Using CSS position, z-index and transform to animate on hover. Outline with outline-offset used for the purple border effect.

Gradients, diagonals and animations

Linear gradient used on header and footer sections and for the h1 element in the main section. The diagonal effect of the header and footer sections is achieved via the use of clip-path. Animations of the heading titles and button are specified in @keyframes rules.
